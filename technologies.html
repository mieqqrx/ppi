Архітектура Платформи

Високопродуктивний стек технологій, створений для масштабування та швидкості.

Frontend

  • Next.js (App Router): Серверний рендеринг (SSR) для миттєвого завантаження сторінок та SEO-оптимізації.
  • TypeScript: Строга типізація для надійності даних відео та коментарів.
  • Redux Toolkit (RTK Query): Ефективне кешування запитів до API, управління стейтом плеєра.
  • Tailwind CSS & Framer Motion: Гнучка стилізація та плавні 60fps анімації взаємодій.

Backend

  • C# / .NET Core 8: Високопродуктивне ядро платформи, створення надійного REST API.
  • Entity Framework Core: Надійна ORM для взаємодії з базою даних.
  • FFmpeg: Конвертація завантажених файлів у HLS-формат (m3u8), розбиття на сегменти, генерація прев'юшок та витягування метаданих (тривалість відео).
  • JWT Authentication: Захищений доступ до ресурсів та системи коментування.
